Boxes

Boxes are the basic building blocks of our UI for prototypes. For the most part, everything will go inside of a box.

This is our most basic box.

      <div class="box"> … </div>

Box heading

Sometimes you want a box that has a heading like this.

<div class="box">
  <header>
    <h4>Box heading</h4>
  </header>
  …
</div>

And sometimes you want a box that has a footer like this...

Box footer

<div class="box">
  …
  <footer>
    <p>Box footer</p>
  </footer>
</div>

If there was a use case where you needed to have the button area in combination with a separate footer, you can always use the thick horizontal rule to create the same effect.
